Smart City Data
Strategy

To help overcome the barriers of formation of a robust data culture, DataSmart Cities Strategy was conceptualized. It covers the building blocks of data culture like data governance, lifecycle of data, data control, data classification and outlines the roles of data in emerging technologies.

Data Maturity Assessment Framework

In the path towards success of Open Data initiative by MoHUA, Data Maturity Assessment Framework aims to build a healthy competitive spirit between cities by conducting activities and cities will be scored and graded basis their performance. The framework covers 2 aspects i.e. the Policy and Governance pillar and the Sector pillar.
